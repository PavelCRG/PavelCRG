# assets/

Статичные SVG для оформления профиля. Для каждого элемента есть пара **dark** / **\*-light.svg**. README показывает нужную через `#gh-dark-mode-only` и `#gh-light-mode-only` (тема GitHub).

Карточки **stats / streak / languages** — в `stats-output/` и `languages-output/`, обновляются Actions из публичных репозиториев GitHub.
